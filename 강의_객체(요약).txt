강사 
	- 지식 - 특성   -- 속성, 변수
	- 가르친다 - 행위  -- 함수, 메서드
	
학생 
	- 배우고자 하는 열정 - 특성 
	- 배우고있다           - 행위 
	
객체 - 속성과 메서드로 구성된 자료형

자바스크립트에서 객체를 정의하는 방법
{
	속성 : 값,
	속성 : 값,
	속성 : 값, 
}

값 - 원시타입 값(숫자, 문자, null, undefined) + 객체(+ 함수 객체)


객체의 속성을 접근하는 방법
 - 마침표 연산자(.)
 
 객체변수명.속성명
 객체변수명.속성명();
 객체변수명.속성명(값, 값..);
 
 - 대괄호 연산자  객체변수명['속성명']
 
 
 
 객체의 속성을 변경 
 - 이미 정의되어 있는 속성에 값을 대입 -> 변경이 된다
 
 객체의 속성을 추가 하는 방법
 - 없는 속성에 값을 대입
 
 객체의 속성을 삭제
 delete 연산자
 
 delete 객체변수명.속성명;
 
 변수를 선언 --> window 객체의 하위 속성
 --> 변수의 속성은 객체의 속성과 동일
 
 없는 속성에 접근 -> 기본값이 undefined
 속성명 -> 변수의 명명 규칙
		-> 따옴표로 감싸주면 명명 규칙 관련 없이 설정
 
 
 const person = {};
 const person = new Object(); -- 내부적으로 Object 생성자에 의해서 생성되어 있다

	마우스 관련 이벤트
	mouseenter - 마우스 커서가 요소에 들어 갔을 때 (본인 요소)
	mouseleave - 마우스 커서가 요소에서 나갔을 때  (본인 요소)

	mouseover - 마우스 커서가 요소에 들어 갔을 때 (본인 + 자식 요소)
	mouseout - 마우스 커서가 요서에서 나갔을 때 (본인 + 자식 요소)
	
	mousedown - 마우스 버튼을 눌렀을때 
	mouseup - 마우스 버튼을 눌렀다 뗄때
	
	mousemove - 마우스 커서를 움직일때
	
	mousewheel - 마우스 휠 버튼을 조작할때 


   키 이벤트
   keydown - 키보드 키를 눌렀을 때  - 한번만 실행 - 거의 모든키 (O)
   keypress  - 키보드 키를 눌렀을 때 - 키를 누르고 있는 동안 계속 실행 
				  (Ctrl, Alt, Shift 는 동작하지 않습니다.)
   
   keyup - 키보드 키를 눌렀다 뗐을 때(OO)
   
   기타
   scroll - 창을 스크롤 했을 때 발생
   resize - 창의 사이즈를 줄였을때 
   
   focus - 입력 항목에 커서가 가 있을때 
   blur - 입력 항목에서 커서가 제거 되었을 때
   
   change - select 태그에서 선택을 했을때, file을 선택 했을때

1. 산술연산자
2. 문자 결합 연산자(+)
3. 대입 연산자(=)
	- 복합 대입 연산자
	(산술 + 대입 연산)
	var a = a + 2; // a += 2;
	
4. 증감 연산자(++a, a++ ..., --a, a--)
5. 비교 연산자(크다,작다, 크거나 같다, 같다, 다르다 ...) -> 논리값을 결과로 반환(true, false)
6. 논리연산자 ( || - OR, && - AND, ! - NOT)		
		(논리값 반환 X)
		var a = 0 || 10; -> true가 되는 조건은 10이므로 10을 반환
		var a = 0 || ""; -> false일지라도 마지막 조건 값인 ""을 반환
7. 삼항 조건 연산자
	(조건식)  ? "참일때" : "거짓일때";
	
	단항(항이 1개) - 증감, 복합 대입 연산자... 
	이항(항이 2개) - a + b

Math
	수학과 관련된 생성자
	.abs() -> 절대값
				음수 - 양수
				양수 - 양수
				
	기준 - 소수점 첫자리
	.round()  -> 반올림
	.floor()   -> 버림
	.ceil()     -> 올림 
	
	.max(...숫자) -> 최대값
	.min(...숫자)  -> 최소값 
	.pow(x, y) -> x의 y 제곱
	.sign() --> 부호 - 음수(-1), 양수(1), 0 - 0
	.sqrt -> root
	.random() -> 난수(무작위 수)를 발생시키는 메서드(0~1사이의 무작위 수)
